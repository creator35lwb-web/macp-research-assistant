# Competitive Landscape Research — MACP Research Assistant (Jun 2026)

## Direct Competitors (Academic Research AI)

### Tier 1: Established Players (Well-funded, large user bases)

1. **Elicit** — 138M papers, structured data extraction, $12/mo Plus
   - Single model (proprietary), citation-backed
   - No multi-model consensus
   - Strong at systematic reviews

2. **Consensus** — Peer-reviewed literature search, Consensus Meter
   - Free tier + paid plans
   - Single model, evidence visualization
   - No BYOK, no multi-agent

3. **Semantic Scholar** (AI2) — 200M+ papers, free
   - Recommendation engine, citation graphs
   - No AI analysis layer (just discovery)
   - Free but limited AI features

4. **Undermind** — Agentic multi-hop paper search, $16-20/mo Pro
   - Highest "agenticness" score (14/36)
   - Reads papers like a PhD student
   - Single model, no consensus

5. **SciSpace** — Literature review, data extraction, 10/36 agenticness
   - Paper chat, citation-backed
   - Single model approach

### Tier 2: Multi-Model Platforms (Closest to MACP concept)

6. **Suprmind** — 5 models (GPT, Claude, Gemini, Grok, Perplexity) debate
   - $19/mo starting, enterprise tiers
   - Sequential/Debate/Red Team/Research Symphony modes
   - 1,324 conversations measured, 3,484 unique insights
   - Knowledge Graph + Vector File Database
   - CLOSEST COMPETITOR to MACP's multi-model consensus
   - BUT: Not open-source, not BYOK, not research-specific (general purpose)

7. **Poe** — Multi-model chat, $20/mo
   - Access to multiple models
   - No consensus/debate mechanism
   - General purpose, not research-focused

### Tier 3: Open Source / Research-Specific

8. **STORM by Stanford** — Wikipedia-style report generation
   - Open source
   - Single model, no consensus
   - Report generation, not ongoing research

9. **ResearchRabbit** — Free, citation network discovery
   - Visualization of paper connections
   - No AI analysis, just discovery

10. **NotebookLM (Google)** — Free, document analysis
    - Single model (Gemini), no multi-agent
    - Good for personal knowledge management

## Market Gap MACP Fills

1. No other tool combines multi-model consensus + BYOK + open source
2. No other tool has agent-to-agent submission (A2A) for research
3. No other tool is MCP-native (Model Context Protocol)
4. Suprmind is closest but is closed-source, general-purpose, $19+/mo
5. Academic tools (Elicit, Consensus) are single-model, no privacy guarantee

## Weaknesses vs Competitors

1. Paper corpus: 12,800 vs Elicit's 138M or Semantic Scholar's 200M+
2. Agenticness: Not yet rated on Agentic.ai rubric
3. User base: 55 active-week EA vs millions for Elicit/Consensus
4. No systematic review workflow (Elicit's strength)
5. No Consensus Meter equivalent (visual evidence strength)
6. Requires API keys for full functionality (BYOK barrier)
